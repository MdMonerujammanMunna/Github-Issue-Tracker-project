1️⃣ What is the difference between var, let, and const?
var, let, and const are keywords used to declare variables in JavaScript.
var is a function scope. As a result, var can be accessed from outside the block, and the same variable name can be declared multiple times using var.
let is a block scope. It cannot be accessed from outside the block. The name given to a variable using let cannot be declared again, but its value can be changed.
const is a block scope. It cannot be accessed from outside the block. The name given to a variable using const cannot be declared again, and its value cannot be changed.

2️⃣ What is the spread operator (...)?
The spread operator is a special operator that is used to spread the elements of an array or object separately.

3️⃣ What is the difference between map(), filter(), and forEach()?
map() is used to update the elements of an array and returns a new array.
filter() is used to select elements from an array based on a condition and returns a new array.
forEach() is used to execute a loop and does not return anything.

4️⃣ What is an arrow function?
An arrow function is a short form of declaring a normal function. It is an ES6 feature that makes writing functions easier and faster.

5️⃣ What are template literals?
Template literals are a special feature of JavaScript. It is used to easily write strings and also easily insert variables inside the string. It is written using backticks (``), and variables are inserted inside the backticks using ${}.
